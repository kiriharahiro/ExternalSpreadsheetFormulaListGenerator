# 📊 外部スプレッドシート 数式一覧化ツール

このツールは、同じスプレッドシート内だけでなく、**「設定表」に指定した別のスプレッドシート（外部ファイル）**からすべての数式（関数）を自動で読み取って一覧化する便利なツールです。

---

## 🚀 導入手順

1. **スプレッドシート「数式一覧作成」の Apps Script を開く**
   * スプレッドシート「数式一覧作成」を開きます。
   * メニューバーの **「拡張機能」 ＞ 「Apps Script」** を選択します。
2. **コードの貼り付け**
   * 最初から入っているコード（もしあれば）をすべて消去します。
   * 本フォルダの [Code.gs](./Code.gs) の中身をコピーし、エディタにすべて貼り付けます。
   * 画面上部の **「保存（フロッピーのアイコン）」** をクリックして保存します。
3. **スプレッドシートの再読み込み**
   * スプレッドシートのタブに戻り、ブラウザの **「再読み込み（リロード）」** を行います。
   * メニューバーのヘルプの右隣付近に **「📊 数式一覧化ツール」** というメニューが追加されます。

---

## 🛠️ 使い方

1. **設定の入力**
   * 「設定表」シートを開きます。
   * **C4 セル**: 調べたい対象のスプレッドシートIDを入力します。
     *(例: `YOUR_SPREADSHEET_ID`)*
   * **C5 セル**: 
     * すべてのシートをスキャンしたい場合は **`*ALL`** と入力（または空欄）にします。
     * 特定のシートだけをスキャンしたい場合は、そのシート名を入力します。C6セル以降に並べることで、複数のシートを同時にスキャン対象にすることもできます。
2. **ツールの実行**
   * メニューバーから **「📊 数式一覧化ツール」 ＞ 「設定表に基づいて数式を一覧化する」** をクリックします。
   * 初回実行時のみ、「承認が必要です」という承認画面が表示されます。画面の指示に従って実行を許可してください。
3. **結果の確認**
   * 処理が完了すると、自動的に **「数式一覧」** という名前の新しいシートが本スプレッドシート内に作成（または上書き）され、対象スプレッドシートから抽出された数式が一覧表示されます。
